SOBRE MIM

Meu nome é André Victor e tive minha primeira experiência com programação em 2016. Comecei aprendendo C e Java que eram tecnologias obrigatórias do meu curso técnico e após me formar dei continuidade desenvolvendo aplicações Desktop em Java e manipulando bancos de dados no ensino superior. Foi quando surgiu o interesse em programação Web e comecei meus estudos por conta em Javascript, CSS e HTML. Logo abaixo cito algumas das tecnologias que tenho familiaridade, mas também existem algumas outras que tive passagem e que também estou estudando. Hoje estou no 4º período de Análise e Desenvolvimento de Sistemas e agora busco experiências no mercado de trabalho de front-end.
