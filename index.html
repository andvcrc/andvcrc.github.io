SOBRE MIM

Meu nome é André Victor e tive minha primeira experiência com programação em 2016. Comecei aprendendo C e Java que eram tecnologias obrigatórias do meu curso técnico, e, após me formar, dei continuidade desenvolvendo aplicações mais complexas que envolviam servidores e bancos de dados. Nisso surgiu o interesse em programação Web e comecei meus estudos por conta em Javascript, CSS e HTML. Logo abaixo cito algumas das tecnologias que tenho familiaridade, mas também existem algumas outras que tive passagem ou que estou estudando. Hoje estou no 4º período de Análise e Desenvolvimento de Sistemas e agora estou estudando e em busca de experiências no mercado de trabalho Full stack e também em projetos e parcerias.

Vamos trabalhar juntos?

Java

Desenvolvi aplicações Java WEB e Desktop, orientadas a objeto, utilizando o Java pelos frameworks do Netbeans, Eclipse e também do Spring, tanto em atividades e projetos de cursos quanto por projetos individuais.

Javascript

Aprendi Javascript no curso superior e também procurei por formações adicionais, onde desenvolvi diversos projetos e funcionalidades que também estão presentes neste site.

CSS

CSS esteve presente como tema dos diversos cursos que fiz, além do superior. Porém não me contive com apenas o que aprendi, logo pesquisei e busquei conhecimento em diversas páginas, fórums e também diversos vídeos e dicas sobre estilizações e manipuações de elementos.

Git e GitHub

Tive prática com Git na faculdade/cursos e também com alguns projetos por conta própria, além de que hospedo todos os meus projetos, seja em andamento ou já finalizados, no próprio Github.

HTML5

HTML é grande parte dos meus projetos atuais. Obtive meu conhecimento em cursos, fórums e dicas de terceiros além de me inspirar em grandes sites e portais da internet. Uso todo esse conhecimento em conjunto com o que sei de tratamento e criação de imagens.

Linguagem C

Linguagem C aprendi no curso técnico e também na faculdade. Desenvolvi diversos projetos desktop de ampla abrangência da linguagem, que serviram como base para meus conhecimentos em todas as linguagens que tenho conhecimento atualmente.

NodeJS

NodeJS é uma das tecnologias que aprendi recentemente. Desenvolvi apenas atividades de curso, utilizando o Node em conjunto com Express. Tenho em mente futuros projetos mais complexos, porém nenhum projeto publicado no momento.

Spring Boot

Atualmente estou aprimorando meus conhecimentos em Spring Boot simultâneo com Java. Tenho um projeto em andamento e uma das branches contém minha API que retorna funções de uma calculadora.
